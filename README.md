# Barony Scroll Tracker

A fan-made web app that creates a database to conveniently track the discovery of scrolls in the video game **Barony**

## Features

- Create a database of scrolls for each character
- The database is pre-populated with scroll labels
- Record scroll effects as you discover them during a run

## Tech Stack

- Next.js
- TypeScript
- React
- PostgreSQL
- Prisma
- Vercel
